Odds is an Android app that allows users to create and vote on the probability of anything. It is designed for Android 5.0+ and built using MVP architecture, Retrofit, Dagger, and RxJava among other modern Android tools. Backend consists of Google Cloud Functions (NodeJS) and Firebase.

SNAP Test is a SNAP (Food Stamp) determination app that enables users to check whether their household may be eligible for SNAP benefits according to TN and USDA policies. Full-stack web app using AngularJS and Materialize for the client; NodeJS, Express, and MongoDB with Mongoose ORM for back-end.
